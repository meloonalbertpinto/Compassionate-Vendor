## 🌟 Overview

**Compassionate Vendor** is an innovative retail automation system designed to bridge the gap between AI and human interaction in shopping. By leveraging real-time computer vision, the system identifies items, manages carts autonomously, and provides a seamless, "compassionate" checkout experience.

> [!NOTE]
> This project was developed as part of the **Project Based Learning (PBL)** curriculum, focusing on integrating backend AI models with a modern frontend interface.

## 🚀 Key Features

<details open>
<summary><b>🔍 Real-time Object Detection</b></summary>
Utilizes MediaPipe and OpenCV to identify fruits and products instantly through the camera feed.
</details>

<details>
<summary><b>🛒 Autonomous Cart Management</b></summary>
Automatically adds detected items to the user's virtual cart without manual input.
</details>

<details>
<summary><b>💳 Seamless Payment Integration</b></summary>
A modern, intuitive interface for finalizing transactions and viewing summaries.
</details>

<details>
<summary><b>📊 Data Driven Insights</b></summary>
Backend logging and training scripts for continuous model improvement.
</details>

## 🛠️ Setup & Installation

### Backend Setup
1. Navigate to the `backend` directory:
   ```bash
   cd CompassionateVendor/backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the server:
   ```bash
   python main.py
   ```

### Frontend Setup
1. Navigate to the `frontend` directory:
   ```bash
   cd CompassionateVendor/frontend
   ```
2. Install packages:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
